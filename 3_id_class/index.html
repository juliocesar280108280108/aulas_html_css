id e class

class

existe duas caracteristicas pricipais do class

  • e possivel utilizar o mesmo class em varios elementos diferentes
  • e possivel ultilizar varias classes no mesmo elementos

id

existem duas caracteristicaspricipais do id que sao contrate com as caracteristicas do class

  • nao e possivel utilizar o mesmo id em varios elementos diferentes
  • nao e possivel utilizar varios ids no mesmo elementos

o id funciona como um cpf. nao podem existir dois iguais e a mesma pessoa nao pode ter varios cpfs
